# 預設快捷鍵  
Ctrl+F2 開啟設定與重設API資料按鈕  
Ctrl+D 查價  
F2 邀請最近密的人進組隊  
F3 %TY  
F4 退組隊，需在設定中設定角色名稱  
F5 回藏身處
